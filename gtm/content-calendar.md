# Content calendar — first 100 users (4-week rollout)

A concrete, repeatable cadence for the [growth strategy](../wiki/spec/growth-strategy.md).
One founder, two personal accounts (X + Reddit), zero ad spend. Pair every post
with a real visual and drive to the waitlist.

> **Posting rhythm:** X ~daily (it rewards consistency), Reddit 3–4 high-value
> posts/week (it punishes volume). Quality and genuine help beat frequency.

## Week 0 — Foundation

| Do | Why |
|---|---|
| Warm network: get 5–10 friends/industry contacts using the tool | First real feedback + first testimonials |
| Set up the personal X account + Reddit account (or warm up existing) | One real account per platform; build karma before posting links |
| Ship the waitlist landing page (`gtm/landing/`) | Every piece of content needs a destination |
| Generate 15–20 showcase visuals to post from | Never post a hook without proof |
| Run the agent once to pull this week's pain points | Content angles come from real threads, not guesses |

## Weeks 1–4 — Distribute, capture, convert, learn

Repeat this weekly shape, swapping in fresh hooks from
[`hooks-bank.md`](./hooks-bank.md):

| Day | X (personal) | Reddit (personal) |
|---|---|---|
| Mon | Build-in-public update + 1 visual | — |
| Tue | Show-don't-tell (before/after) | Help-first reply in a `help_only` sub (no product) |
| Wed | Pain-point thread 🧵 | — |
| Thu | Style showcase / "X covers in 2 min" | Story-framed show-and-tell in an `open` sub (r/musicmarketing) |
| Fri | Contrarian take (earns replies) | Curiosity/question post (AEO target) |
| Sat | Repost best-performing + reply to comments | Genuine help reply (no product) |
| Sun | Light: 1 visual, no ask | — |

**Each week also:** run the agent, refresh the top pain points, and log every new
signup's source ("how did you hear about us?").

## Per-post checklist (run before every post)

- [ ] **Helps first** — useful even if nobody clicks the link?
- [ ] **Right policy** — product mention only in `open`/X; pure help in `help_only`/`strict`?
- [ ] **Discloses** — if I mention SoundCave, am I open that I'm building it?
- [ ] **Has proof** — paired with a real visual?
- [ ] **Drives to waitlist** — soft CTA at the end, not the start?
- [ ] **On-voice** — plain, specific, warm, no hype (see `src/prompts/draft.md`)?
- [ ] **Not spammy** — one account, human-posted, no link blasting?

## What to measure (weekly)

- Posts shipped vs. planned
- Impressions, profile visits, link clicks → landing
- Waitlist signups + **source breakdown**
- Which hooks/communities/segments convert → double down next week

**Target:** 100 waitlist signups. When a channel or hook type clearly wins, pour
more in and cut the rest — that's the 100 → 1,000 lever.
